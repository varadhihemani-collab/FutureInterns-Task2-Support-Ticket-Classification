# Support Ticket Classification System

## Project Overview
This project aims to build an AI-based Support Ticket Classification System that automatically classifies customer support tickets into different categories and helps support teams handle customer issues faster and more efficiently.

## Objective
To develop a Machine Learning model that analyzes customer complaints, understands the text data, and predicts the appropriate ticket category.

## Technologies Used
- Python
- Google Colab / Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- NLP (Natural Language Processing)
- TF-IDF Vectorization

## Key Features
✔ Text cleaning and preprocessing  
✔ Text tokenization  
✔ Feature extraction using TF-IDF  
✔ Automatic ticket category classification  
✔ Priority tagging (High / Medium / Low)  
✔ Model performance evaluation  
✔ Prediction on new customer complaints  

## Project Workflow
1. Load and analyze the support ticket dataset.
2. Clean and preprocess customer complaint text.
3. Convert text into numerical features using TF-IDF.
4. Train a Machine Learning classification model.
5. Evaluate model performance.
6. Predict categories for new support tickets.

## Machine Learning Approach
The project uses Natural Language Processing techniques and Machine Learning classification algorithms to understand ticket descriptions and classify them automatically.

## Benefits
- Reduces manual ticket sorting effort.
- Helps support teams respond faster.
- Improves customer service efficiency.
- Automates ticket management.

## How to Run the Project
1. Open the notebook in Google Colab.
2. Install required libraries.
3. Upload the dataset.
4. Run all cells step-by-step.
5. Enter a customer complaint to get the predicted category.
